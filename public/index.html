🧩 Cloud-Native Microservices Architecture — Live Demo

Reference implementation accompanying the IEEE paper. Auth, Product, and Order services running independently behind an API Gateway.

⏳ Hosted on a free-tier server that sleeps after inactivity — the first click below may take 30–60 seconds to respond while it wakes up. Every click after that is fast.

1. Login (Auth Service)

2. Product Catalog (Product Service)

3. Place an Order (Order → Product inter-service call)

4. System health

Source code, benchmarks, Kubernetes/Terraform deployment configs, and the full paper are in the GitHub repository.
